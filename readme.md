# ani-subs

可用于 [Animeko](https://github.com/open-ani/animeko) 的数据源订阅。

## 用法

设置中添加订阅：

- `https://sub.creamycake.org/v1/css1.json`
- `https://sub.creamycake.org/v1/bt1.json`

支持 Ani 4.0.0 及以上版本。
